# 수목원

이수목이 스스로 공부한 개념과 실습 기록을 모아두는 개인 위키입니다.

## 운영 원칙

- `main`은 문서 사이트와 학습 흐름의 기준 브랜치로 유지합니다.
- 실습은 `lab/{topic}` 브랜치에서 진행합니다.
- 실행해본 예제 중 문서화할 가치가 있는 내용만 `main`의 `docs/`로 옮깁니다.
- 문서는 결론, 이유, 재현 방법, 참고 코드 순서로 짧게 남깁니다.

## 큰 가지

### CS

- 자료구조

- 운영체제
- 네트워크
- 데이터베이스
- 컴퓨터 구조

### AI

- 딥러닝
