# 임베딩

**임베딩(embedding)** 은 토큰, 문장, 이미지 같은 대상을 모델이 계산할 수 있는 숫자 벡터로 표현하는 방법입니다.

토크나이저가 토큰을 정수 ID로 바꾸면, 모델은 각 ID를 여러 숫자로 이루어진 벡터로 변환합니다. 이 벡터의 값은 사람이 직접 정하는 것이 아니라 학습 과정에서 조정됩니다.

```text
토큰 "고양이" → 토큰 ID → [0.12, -0.37, 0.81, ...]
```

학습이 진행되면 비슷한 문맥에서 사용되는 대상은 벡터 공간에서도 관련된 위치와 방향을 갖게 됩니다. 모델은 이 숫자 관계를 이용해 토큰 사이의 유사성과 문맥을 계산합니다.

입력 단계의 토큰 임베딩만으로 단어의 의미가 완성되는 것은 아닙니다. 같은 단어도 주변 문장에 따라 뜻이 달라질 수 있으며, Transformer의 여러 층을 지나면서 현재 문맥이 반영된 표현으로 계속 바뀝니다.
