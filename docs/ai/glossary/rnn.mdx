# RNN

**RNN(Recurrent Neural Network, 순환 신경망)** 은 문장처럼 순서가 있는 데이터를 앞에서부터 차례대로 처리하는 신경망입니다.

RNN은 현재 입력과 이전 단계에서 전달받은 **은닉 상태(hidden state)** 를 함께 사용합니다. 은닉 상태는 지금까지 읽은 내용의 요약 정보처럼 동작합니다.

```text
현재 단어 + 이전 은닉 상태 → 새로운 은닉 상태 → 다음 단계
```

문장의 순서를 반영할 수 있다는 장점이 있지만, 문장이 길어지면 앞부분의 정보가 약해질 수 있습니다. 토큰을 순서대로 처리해야 하므로 대규모 병렬 학습에도 불리합니다.

LSTM과 GRU는 필요한 정보를 더 오래 유지하도록 RNN을 개선한 구조입니다. 이후에는 병렬 처리와 긴 문맥 처리에 유리한 Transformer가 대규모 언어 모델의 중심 구조가 되었습니다.
