# 어텐션

**어텐션(attention)** 은 현재 토큰을 처리할 때 문장 안의 다른 토큰을 얼마나 참고할지 계산하는 방법입니다.

예를 들어 `철수는 사과를 샀고, 그는 그것을 먹었다`에서 `그`는 `철수`와, `그것`은 `사과`와 관련이 큽니다. 어텐션은 이런 토큰 사이의 관련성에 서로 다른 가중치를 부여합니다.

Transformer의 **셀프 어텐션(self-attention)** 에서는 같은 문장 안의 토큰들이 서로를 참고합니다. 각 토큰에서 만든 Query, Key, Value 벡터를 이용해 관련성 점수를 계산하고, 점수가 높은 정보에 더 큰 비중을 둡니다.

어텐션 덕분에 모델은 멀리 떨어진 토큰의 관계를 비교적 효과적으로 다룰 수 있습니다. 다만 어텐션 점수가 곧 사람이 이해하는 설명이나 인과관계를 의미하는 것은 아닙니다.
