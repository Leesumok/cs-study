# 멀티모달

**멀티모달(multimodal)** 은 텍스트, 이미지, 음성, 영상처럼 서로 다른 형태의 데이터를 함께 처리하는 능력이나 모델을 뜻합니다. 여기서 각 데이터 형태를 **모달리티(modality)** 라고 부릅니다.

예를 들어 멀티모달 모델은 다음과 같은 작업을 할 수 있습니다.

- 사진을 보고 내용을 글로 설명하기
- 문서 이미지에서 표와 글자를 읽고 질문에 답하기
- 음성을 듣고 텍스트로 옮기거나 대답하기
- 텍스트 설명을 바탕으로 이미지 만들기

모델이나 시스템은 각 모달리티를 숫자 표현으로 변환하고, 서로 관련된 정보를 연결해 결과를 만듭니다. 모든 멀티모달 모델이 모든 입력과 출력을 지원하는 것은 아니며, 지원 범위는 모델마다 다릅니다.

이미지나 음성을 처리할 수 있어도 내용을 항상 정확히 이해하는 것은 아닙니다. 작은 글자, 복잡한 공간 관계, 음질이 낮은 녹음처럼 입력이 불명확한 경우에는 별도의 검증이 필요합니다.
