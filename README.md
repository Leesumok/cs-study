# cs-study

Computer science study notes and Kotlin practice.

## 공부 방식

이 저장소는 CS 학습 내용을 브랜치별로 남기기 위한 공간입니다.

- `main`: 저장소 소개와 전체 학습 흐름
- 주제별 브랜치: 각 CS 주제의 정리, 예제 코드, 실습 기록
- 언어: Kotlin

## 예상 주제

- 자료구조
- 알고리즘
- 운영체제
- 네트워크
- 데이터베이스
- 컴퓨터 구조

## 기록 규칙

각 브랜치에는 주제별 README, Kotlin 예제 코드, 풀이 또는 실험 기록을 함께 남깁니다.
